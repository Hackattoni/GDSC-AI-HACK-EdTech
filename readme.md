**ListenUp** is an AI-powered educational tool designed to help students improve their oral presentations. The system analyzes student audio recordings, compares them against a reference text (PDF), and automatically flags conceptual inaccuracies or inconsistencies.

## üöÄ Key Features
* **Audio Analysis:** Automatic transcription of the student's speech using AI.
* **Smart Matching:** Semantic comparison between the spoken transcript and the study material (PDF).
* **Fact-Checking:** Identification of errors or missing concepts in the student's explanation.
